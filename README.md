# html2gutenberg
Simple and basic PHP function to "convert" HTML to WordPress Gutenberg markup. No code optimization, it just works !

HTML tags included :
- Paragraphs (p)
- Titles (H1 to H6)
- Images (img)
- Citations (blockquote)
- Unordered lists (ul / li)
- Ordered lists (ol / li)
- Arrays (table)
